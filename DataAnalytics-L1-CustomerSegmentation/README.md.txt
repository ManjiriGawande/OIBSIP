# Customer Segmentation Analysis using K-Means Clustering

## Oasis Infobyte Internship

### Project Overview
This project performs Customer Segmentation using the Online Retail dataset.
The objective is to identify different groups of customers based on their purchasing behavior using RFM Analysis (Recency, Frequency, Monetary) and K-Means Clustering.

### Objectives
- Perform data cleaning
- Handle missing values
- Perform Exploratory Data Analysis (EDA)
- Calculate RFM metrics
- Standardize data
- Apply K-Means clustering
- Visualize customer segments
- Generate business insights

### Technologies Used
- Python
- Pandas
- NumPy

Generated business insights for:

- Loyal Customers
