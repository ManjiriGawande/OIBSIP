This repository contains the projects completed during my **Data Analytics Internship** at **Oasis Infobyte**. These projects focus on data cleaning, exploratory data analysis (EDA), machine learning, customer analytics, predictive modeling, and business intelligence using Python.

### Projects

## 1️⃣ Customer Segmentation Analysis using K-Means Clustering

* Overview
Performed customer segmentation using the Online Retail dataset by applying RFM (Recency, Frequency, Monetary) Analysis and K-Means Clustering to identify distinct customer groups for targeted marketing strategies.

* Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- RFM Analysis
- StandardScaler
- Elbow Method
- K-Means Clustering
- Customer Segmentation
- Cluster Visualization
- Business Recommendations

* Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
